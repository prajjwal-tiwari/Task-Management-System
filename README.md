# Task Management System V2

## Overview

Workflow based task management platform developed for manufacturing and NPI teams.

## Features

- Task Assignment
- Approval Workflow
- Employee Submission Portal
- Due Date Monitoring
- Email Notifications
- Reminder Management
- Project Tracking
- Overdue Monitoring

## Technology Stack

- Google Apps Script
- Google Sheets
- Gmail Integration
- Google Drive

## Business Impact

Reduced manual follow-up activities and improved task visibility across teams.
